Badge is a small uppercase status pill. Tones map to brand/semantic colors; `solid` fills it.

```jsx
<Badge tone="gold">Hundo</Badge>
<Badge tone="green" solid>PvP</Badge>
<Badge tone="neutral">Lvl 50</Badge>
```

Tones: `neutral | gold | blue | red | green`. Display font, tracked + uppercased. For league-rank quality use the dedicated `RankBadge`.
